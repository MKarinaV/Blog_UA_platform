# BlogUA - Блогова Платформа

Flask + SQLite веб-застосунок з повним функціоналом блогу.Платформа підтримує систему ролей, модерацію коментарів, створення постів та базову систему адміністрування.

## Встановлення та запуск

Спочатку потрібно встановити необхідні залежності:

pip install -r requirements.txt

Після цього запустити сервер:

python app.py

Після запуску застосунок буде доступний у браузері за адресою:

http://localhost:5000

## Реалізований функціонал

### 1. Реєстрація та автентифікація
- Реєстрація з валідацією (ім'я, email, пароль)
- Вхід / вихід із сесіями Flask
- Безпечне хешування паролів через werkzeug.security
- Захист від входу заблокованих акаунтів

### 2. Управління постами
- Створення, редагування, видалення постів
- Статуси: **Чернетка** / **Опублікований**
- Теги, лічильник переглядів
- Автофільтрація заборонених слів у контенті

### 3. Коментування
- Залишення коментарів до постів
- Коментарі від звичайних користувачів **потребують схвалення**
- Коментарі адміністратора публікуються відразу
- Видалення власних коментарів

### 4. Система ролей і модерація
| Роль        | Можливості |
|-------------|-----------|
| `user`      | Читання, написання постів та коментарів |
| `moderator` | Те саме + модерація коментарів (схвалення/відхилення) |
| `admin`     | Повний доступ: управління користувачами, постами, коментарями |

- Блокування / розблокування користувачів
- Зміна ролей користувачів
- Видалення акаунтів (з постами та коментарями)
- Схвалення / відхилення коментарів
- Видалення будь-яких постів та коментарів
- Автоматична фільтрація слів-заборон

### Тестування

Для запуску тестів потрібно виконати:

py -m pytest

тести перевіряють:

відкриття головної сторінки
доступність сторінки входу
доступність сторінки реєстрації
захист адміністративної панелі від неавторизованого доступу.
