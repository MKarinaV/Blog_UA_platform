# BlogUA — Блогова Платформа

Flask + SQLite веб-застосунок з повним функціоналом блогу.

## Встановлення та запуск

```bash
# 1. Встановити залежності
pip install flask

# 2. Запустити сервер
python app.py
```

Відкрийте браузер: **http://localhost:5000**

---

## Акаунт адміністратора (за замовчуванням)

| Поле     | Значення   |
|----------|------------|
| Логін    | `admin`    |
| Пароль   | `admin123` |

## Структура проекту

```
blog/
├── app.py              # Головний Flask-застосунок (маршрути)
├── database.py         # Ініціалізація БД та підключення
├── blog.db             # SQLite база даних (створюється автоматично)
├── requirements.txt    # Залежності
└── templates/
    ├── base.html       # Базовий шаблон (навігація, стилі)
    ├── index.html      # Головна сторінка (стрічка постів)
    ├── login.html      # Сторінка входу
    ├── register.html   # Реєстрація
    ├── post.html       # Перегляд посту + коментарі
    ├── post_form.html  # Форма створення/редагування посту
    ├── profile.html    # Профіль користувача
    ├── my_posts.html   # Мої пости
    └── admin/
        ├── panel.html    # Адмін-панель (огляд + статистика)
        ├── users.html    # Управління користувачами
        ├── posts.html    # Управління постами
        └── comments.html # Модерація коментарів
```

## Реалізований функціонал

### 1. Реєстрація та автентифікація
- Реєстрація з валідацією (ім'я, email, пароль)
- Вхід / вихід із сесіями Flask
- Хешування паролів (SHA-256)
- Захист від входу заблокованих акаунтів

### 2. Управління постами
- Створення, редагування, видалення постів
- Статуси: **Чернетка** / **Опублікований**
- Теги, лічильник переглядів
- Автофільтрація заборонених слів у контенті

### 3. Коментування
- Залишення коментарів до постів
- Коментарі від звичайних користувачів **потребують схвалення**
- Коментарі адміністратора публікуються відразу
- Видалення власних коментарів

### 4. Система ролей і модерація
| Роль        | Можливості |
|-------------|-----------|
| `user`      | Читання, написання постів та коментарів |
| `moderator` | Те саме + підвищений пріоритет |
| `admin`     | Повний доступ: управління користувачами, постами, коментарями |

- Блокування / розблокування користувачів
- Зміна ролей користувачів
- Видалення акаунтів (з постами та коментарями)
- Схвалення / відхилення коментарів
- Видалення будь-яких постів та коментарів
- Автоматична фільтрація слів-заборон

## База даних (SQLite)

```sql
users    (id, username, email, password_hash, role, is_blocked, bio, created_at)
posts    (id, title, content, tags, author_id, status, views, created_at, updated_at)
comments (id, post_id, author_id, content, created_at, is_approved)
```
